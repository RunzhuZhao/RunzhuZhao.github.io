斐波那契数列：后一个数等于前两个数之和；例：1 1 2 3 5 8 13 21 34

```Objective-C
NSInteger a = 1;
NSInteger b = 1;
for (NSInteger i = 0; i < 20; i++) {
    NSLog(@"%@", @(a));
    NSInteger sum = a + b;
    a = b;
    b = sum;
}
```  

```Swift
var a = 1
var b = 1
for _ in 0...20 {
    print(a)
    let sum = a + b
    a = b
    b = sum
}
```
